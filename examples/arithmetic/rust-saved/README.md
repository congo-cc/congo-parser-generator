# rust-saved -- Hand-Written Files for Arithmetic Parsers

This directory contains hand-written Rust files that extend the generated
arithmetic parsers with evaluation logic, CLI binaries, tests, and
documentation.

## Purpose

The files in `rust-arith1/` and `rust-arith2/` are regenerated by
`ant rust-gen` (via `congocc.jar`).  Regeneration overwrites all files,
including hand-written additions like `inject.rs` (evaluate logic), `main.rs`
(CLI binary), and `tests/evaluate.rs`.

This `rust-saved/` directory preserves the hand-written files so they can be
merged back after regeneration.

## After Regenerating Parsers

After running `ant rust-gen` in the arithmetic example directory:

```bash
# Copy saved files back into the generated directories
cp rust-saved/arith2/inject.rs   rust-arith2/inject.rs
cp rust-saved/arith2/main.rs     rust-arith2/main.rs
cp rust-saved/arith2/Cargo.toml  rust-arith2/Cargo.toml
cp rust-saved/arith2/lib.rs      rust-arith2/lib.rs
cp rust-saved/arith2/README.md   rust-arith2/README.md
cp rust-saved/arith2/tests/evaluate.rs rust-arith2/tests/evaluate.rs

cp rust-saved/arith1/inject.rs   rust-arith1/inject.rs
cp rust-saved/arith1/main.rs     rust-arith1/main.rs
cp rust-saved/arith1/Cargo.toml  rust-arith1/Cargo.toml
cp rust-saved/arith1/lib.rs      rust-arith1/lib.rs
cp rust-saved/arith1/README.md   rust-arith1/README.md
cp rust-saved/arith1/tests/evaluate.rs rust-arith1/tests/evaluate.rs
# Verify everything still works
cd rust-arith2 && cargo test
cd ../rust-arith1 && cargo test
```cp rust-saved/arith2/lib.rs      rust-arith2/lib.rs

## File Inventory

### arith2/ (evaluator)

| File | Description |
|------|-------------|
| `inject.rs` | EvalError enum + evaluate() implementation |
| `main.rs` | CLI binary: parse, evaluate, print result |
| `Cargo.toml` | Adds [[bin]] and [[test]] evaluate sections |
| `lib.rs` | Adds `pub use inject::EvalError` re-export |
| `tests/evaluate.rs` | 41 comprehensive evaluation tests |
| `README.md` | Build, run, test, and API documentation |

### arith1/ (parse-only)

| File | Description |
|------|-------------|
| `inject.rs` | EvalError::Unsupported stub evaluate() |
| `main.rs` | CLI binary: parse and dump AST |
| `Cargo.toml` | Adds [[bin]] and [[test]] evaluate sections |
| `lib.rs` | Adds `pub use inject::EvalError` re-export |
| `tests/evaluate.rs` | 8 tests (unsupported + parsing regression) |
| `README.md` | Build, run, test, and API documentation |
